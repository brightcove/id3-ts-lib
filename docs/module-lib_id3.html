(static) id3TagChunked(buffer, destination, startOffset, endOffset, payload)

Generates a specified portion (a "chunk") of an ID3 tag. The tag is hard-coded to contain a single TXXX frame with the payload.
Parameters:
Name Type Description
buffer Buffer A pre-allocated buffer to fill. Allowed to be shorter than the length of the id3 tag
destination Number The offset to begin writing the ID3 tag into the buffer
startOffset Number The absolute offset into the id3-tag from which to start writing to 'buffer'
endOffset Number The absolute offset into the id3-tag from which to stop writing into the 'buffer'
payload String The text that the id3-tag will contain.

(inner) stringToBufferChunked(payload, buffer, destination, startOffset, endOffset)

Copies a portion of a string into a buffer, in place, without allocating any new Buffers.
Parameters:
Name Type Description
payload String The text to convert into a buffer
buffer Buffer A pre-allocated buffer to fill. Allowed to be shorter than the length of the payload but must fit the ranges specified
destination Number The offset into the buffer from which to start writing the 'payload'.
startOffset Number The offset into the payload from which to start writing to 'buffer'.
endOffset Number The offset into the payload from which to stop writing to 'buffer'.
